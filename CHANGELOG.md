## [1.0.0] - 2026-05-01

### Added
- Multi-source health data collection (Garmin Connect, Health Auto Export, QWeather)
- SQLite-based local storage with 20+ tables
- LLM-powered health advisory engine (Claude / Baichuan)
- 9-dimension health assessment models
- Automated daily pipeline with cron support
- Web dashboard (Streamlit) with 9 pages: overview, trends, correlations, predictions, goals, experiments, etc.
- Risk prediction models for uric acid, hypertension, hyperlipidemia, hyperglycemia
- Goal tracking subsystem with progress monitoring
- Feedback loop and strategy learning
- N-of-1 self-experimentation framework
- Appointment reminder system
- Weather-based outdoor activity suitability
- Docker support with docker-compose
- GitHub Actions CI pipeline (multi-Python-version testing, linting, type checking)
